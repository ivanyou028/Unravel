> Runtime agnostic JS utils

## Installation

```sh
# Using npm
npm i std-env

# Using pnpm
pnpm i std-env

# Using yarn
yarn add std-env
```

## Usage

```js
// ESM
import { env, isDevelopment, isProduction } from "std-env";

// CommonJS
const { env, isDevelopment, isProduction } = require("std-env");
```
